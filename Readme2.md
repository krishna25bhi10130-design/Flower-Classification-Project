Project Title

Iris Flower Classification using Machine Learning (Logistic Regression)

📌 Overview of the Project

This project uses the Iris dataset to classify flowers into three species based on sepal and petal measurements. The model is built using Logistic Regression, trained on real scientific data, and tested for accuracy. The goal is to demonstrate a simple yet powerful machine learning workflow.

📌 Features

Loads Iris dataset using scikit-learn

Splits dataset into training and testing

Trains a Logistic Regression model

Predicts the species of a new flower sample

Displays accuracy of the model

Shows dataset preview

📌 Technologies / Tools Used

Python 3

Pandas

Scikit-learn (sklearn)

Logistic Regression

Jupyter Notebook / Google Colab / IDE

📌 Steps to Install & Run the Project

Clone the Repository
git clone

Install Required Libraries
pip install pandas scikit-learn

Run the Python File
python iris_classification.py
